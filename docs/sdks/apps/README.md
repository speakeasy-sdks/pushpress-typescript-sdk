# Apps
(*apps*)

## Overview

### Available Operations

* [list](#list) - List all available apps
* [get](#get) - Get details of a specific app
* [install](#install) - Install an app for a company

## list

List all available apps

### Example Usage

```typescript
import { PushpressTs } from "pushpress-ts";

const pushpressTs = new PushpressTs({
  apiKey: process.env["PUSHPRESSTS_API_KEY"] ?? "",
});

async function run() {
  const result = await pushpressTs.apps.list();

  // Handle the result
  console.log(result);
}

run();
```

## install

  apiKey: process.env["PUSHPRESSTS_API_KEY"] ?? "",
});

  await pushpressTs.apps.install({
    appId: "<id>",
    requestBody: {
      company: "Hirthe and Sons",
    },
