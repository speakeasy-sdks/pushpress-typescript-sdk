# Installs
(*apps.installs*)

## Overview

### Available Operations

* [list](#list) - List all installs of a specific app
* [uninstall](#uninstall) - Soft uninstall an app (mark it as uninstalled but retain data)
* [hardDelete](#harddelete) - Hard delete an app (remove all data)
* [get](#get) - Get details of a specific app install

## list

List all installs of a specific app

### Example Usage

```typescript
import { Pushpress } from "pushpress";

const pushpress = new Pushpress({
  apiKey: process.env["PUSHPRESS_API_KEY"] ?? "",
});

async function run() {
  const result = await pushpress.apps.installs.list({
    appId: "<id>",
  });

  for await (const page of result) {
    // Handle the page
    console.log(page);
  }
}

run();
```
